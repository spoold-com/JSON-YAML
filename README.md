Convert JSON to YAML and YAML to JSON instantly with bidirectional conversion. Auto-detection, syntax highlighting, and real-time formatting.

## Features

- Convert JSON to YAML and YAML to JSON in both directions
- Auto-detects input format so you can paste either
- Syntax highlighting and real-time conversion as you type
- Preserves structure; round-trip without data loss
- Copy or download converted output

## Why use this tool

Switch between JSON configs and YAML (e.g. Kubernetes, CI) or translate API payloads into readable YAML. No plugins or local tools required.
